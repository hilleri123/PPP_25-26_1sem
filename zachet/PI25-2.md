## Вариант 3: «Нормализация словаря слов»

**Описание:**

- Разбить текст на слова (буквы/цифры).
- Вывести: количество уникальных слов, топ‑5 по частоте, длину слова с максимальным числом разных букв.

**Пример ввода:**

```
Hello, world! Hello Python. Python is great, Python!
```

**Пример вывода:**

```
5
python 3
hello 2
world 1
is 1
great 1
6
```

**Пояснение:**

- Уникальные слова (в нижнем регистре): `hello`, `world`, `python`, `is`, `great` — 5 слов
- Частоты: `python`=3, `hello`=2, остальные по 1
- Слово `python` имеет 6 разных букв (максимум)

## Вариант 4: «Локальный максимум в таблице»

**Описание:**

- Локальный максимум — элемент строго больше всех существующих 4‑соседей.
- Вывести: количество локальных максимумов, максимальный из них, координаты первого.

**Пример ввода:**

```
3 4
1 5 2 3
4 9 7 6
2 3 8 1
```

**Пример вывода:**

```
2
9
1 1
```

**Пояснение:**

- Локальные максимумы: `9` (позиция 1,1) и `8` (позиция 2,2)
- Количество: 2
- Максимальное значение: 9
- Первый найденный: строка 1, столбец 1 (0‑based)
